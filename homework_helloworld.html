1. Basic Hello World

Write a Python program that prints "Hello, World!" to the console.

print("Hello, World!")

This code uses the print() function to display the text "Hello, World!" on the console. The print() function is a built-in Python function that outputs text to the console.

2. Personalized Greeting

Modify the program to print a personalized greeting. For example, if your name is John, the output should be "Hello, John!".

name = "John"\nprint(f"Hello, {name}!")

This code uses a variable name to store the name "John". The print() function then uses an f-string to include the value of the name variable in the output.

3. Multiple Lines

Write a Python program that prints the following text on separate lines:

Hello, World!\nWelcome to Python programming.
print("Hello, World!")\nprint("Welcome to Python programming.")

This code uses two print() function calls to print text on separate lines. Each print() function call outputs text followed by a newline character.

4. Using Variables

Store the greeting message in a variable and then print it.

greeting = "Hello, World!"\nprint(greeting)

This code stores the greeting message "Hello, World!" in a variable named greeting. The print() function then outputs the value of the greeting variable.

5. Combining Strings

Write a program that combines two strings and prints the result. For example, combine "Hello" and "World" to print "Hello World".

str1 = "Hello"\nstr2 = "World"\nprint(str1 + " " + str2)

This code combines two strings str1 and str2 using the + operator. The print() function then outputs the combined string.

6. Printing Quotes

Write a Python program that prints a quote. For example, print "The only limit to our realization of tomorrow is our doubts of today." - Franklin D. Roosevelt.

print('"The only limit to our realization of tomorrow is our doubts of today." - Franklin D. Roosevelt')

This code uses the print() function to output a string that includes both single and double quotes. The string is enclosed in double quotes, and the inner double quotes are escaped using a backslash.

7. Escape Characters

Write a Python program that prints a string with a newline character and a tab character. For example:

Hello,\n\tWorld!
print("Hello,\\n\\tWorld!")

This code uses escape characters \\n for newline and \\t for tab. The print() function outputs the string with the escape characters interpreted as a newline and a tab.

8. Printing a Backslash

Write a Python program that prints a string containing a backslash. For example, print "This is a backslash: \\".

print("This is a backslash: \\\\")

This code uses a double backslash \\\\ to print a single backslash. The print() function outputs the string with the backslash included.

9. Formatted Strings

Write a Python program that uses formatted strings to print the result of a calculation. For example, print "The result of 5 + 3 is 8".

result = 5 + 3\nprint(f"The result of 5 + 3 is {result}")

This code uses an f-string to include the result of the calculation 5 + 3 in the output. The print() function outputs the formatted string with the result.

10. Printing a Multiline String

Write a Python program that prints a multiline string using triple quotes. For example:

This is a multiline string.\nIt can span multiple lines.
print("""This is a multiline string.\nIt can span multiple lines.""")

This code uses triple quotes """ to define a multiline string. The print() function outputs the string with the newline character interpreted as a line break.

11. Basic Input

Write a Python program that asks the user for their name and then prints a greeting message. For example:

name = input("What is your name? ")\nprint(f"Hello, {name}!")

This code uses the input() function to ask the user for their name. The print() function then uses an f-string to include the user's name in the greeting message.

12. Numerical Input

Write a Python program that asks the user for a number, adds 10 to it, and then prints the result. For example:

num = int(input("Enter a number: "))\nresult = num + 10\nprint(f"The result is {result}")

This code uses the input() function to ask the user for a number and converts the input to an integer using int(). The print() function then outputs the result of adding 10 to the input number.

13. Even or Odd

Write a Python program that asks the user for a number and then prints whether the number is even or odd. For example:

num = int(input("Enter a number: "))\nif num % 2 == 0:\n    print("The number is even")\nelse:\n    print("The number is odd")

This code uses the input() function to ask the user for a number and converts the input to an integer using int(). The if statement checks if the number is even or odd using the modulus operator %.

14. Remainder Calculation

Write a Python program that asks the user for two numbers and then prints the remainder when the first number is divided by the second number. For example:

num1 = int(input("Enter the first number: "))\nnum2 = int(input("Enter the second number: "))\nremainder = num1 % num2\nprint(f"The remainder is {remainder}")

This code uses the input() function to ask the user for two numbers and converts the inputs to integers using int(). The print() function then outputs the remainder of the division using the modulus operator %.

15. Basic Calculator

num1 = int(input("Enter the first number: "))\nnum2 = int(input("Enter the second number: "))\nsum = num1 + num2\ndifference = num1 - num2\nproduct = num1 * num2\nquotient = num1 / num2\nprint(f"The sum is {sum}")\nprint(f"The difference is {difference}")\nprint(f"The product is {product}")\nprint(f"The quotient is {quotient}")

This code uses the input() function to ask the user for two numbers and converts the inputs to integers using int(). The print() function then outputs the sum, difference, product, and quotient of the numbers.

16. Age in Future

age = int(input("What is your current age? "))\nfuture_age = age + 10\nprint(f"In 10 years, you will be {future_age} years old.")

This code uses the input() function to ask the user for their current age and converts the input to an integer using int(). The print() function then outputs the user's age in 10 years.

17. Temperature Conversion

Write a Python program that asks the user for a temperature in Celsius and then converts it to Fahrenheit. For example:

celsius = float(input("Enter temperature in Celsius: "))\nfahrenheit = (celsius * 9/5) + 32\nprint(f"The temperature in Fahrenheit is {fahrenheit}")

This code uses the input() function to ask the user for a temperature in Celsius and converts the input to a float using float(). The print() function then outputs the temperature in Fahrenheit using the formula (celsius * 9/5) + 32.

18. Area of a Circle

Write a Python program that asks the user for the radius of a circle and then calculates and prints the area of the circle. For example:

import math\nradius = float(input("Enter the radius of the circle: "))\narea = math.pi * radius ** 2\nprint(f"The area of the circle is {area}")

This code uses the input() function to ask the user for the radius of a circle and converts the input to a float using float(). The math.pi constant is used to calculate the area of the circle using the formula math.pi * radius ** 2. The print() function then outputs the area of the circle.

19. Simple Interest

Write a Python program that calculates the simple interest for a given principal amount, rate of interest, and time period. For example:

principal = float(input("Enter the principal amount: "))\nrate = float(input("Enter the rate of interest: "))\ntime = float(input("Enter the time period in years: "))\ninterest = (principal * rate * time) / 100\nprint(f"The simple interest is {interest}")

This code uses the input() function to ask the user for the principal amount, rate of interest, and time period, and converts the inputs to floats using float(). The simple interest is calculated using the formula (principal * rate * time) / 100. The print() function then outputs the simple interest.

20. BMI Calculator

Write a Python program that calculates the Body Mass Index (BMI) for a given weight and height. For example:

weight = float(input("Enter your weight in kilograms: "))\nheight = float(input("Enter your height in meters: "))\nbmi = weight / (height ** 2)\nprint(f"Your BMI is {bmi}")

This code uses the input() function to ask the user for their weight and height, and converts the inputs to floats using float(). The BMI is calculated using the formula weight / (height ** 2). The print() function then outputs the BMI.
